# SamWaf-Custom-Intercept-Response-Interface
SamWaf开源轻量级网站防火墙系统 自定义拦截响应界面

比较适合SamWaf的拦截页面，纯HTML

包含两种主题：白色背景颜色字  和  颜色背景白色字(推荐)  
橙色 #ff9800：400/502/504  
蓝色 #2196F3：401    
红色 #f44336：403/500  
灰色 #9E9E9E：404  
黄色 #FFC107：429/503  

每个页面都保持了：  
统一的视觉设计和布局  
盾牌+感叹号图标设计  
响应式布局  
适当的操作按钮  
底部品牌标识  
请求ID显示区域  

您可以根据需要在SamWaf中配置这些页面，每个页面会自动替换 [[.SAMWAF_REQ_UUID]] 为实际的请求ID。
